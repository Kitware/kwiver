What are these files?

- generic_tracks.kw18: a set of arbitrary kw18 tracks

- generic_tracks.geom.yml: the same tracks, converted to kpf via
  'vpd2kpf-canonical.pl -do G generic_tracks.kw18', found in
   kwgitlab.kitware.com:roddy.collins/diva-scripts.git

Update 20 Sept 2019

The 2018-03-07.16-55-06.17-00-06.school.G336.* files are annotations
on a clip from MEVA KF1 (mevadata.org), which contain the new
long int IDs which break the current reader.
